# hash-table-assignment

Реализация хеш-таблицы и простых хеш-функций на Python.  
Цель проекта — отработать навыки работы с хешированием, обработкой коллизий и динамическим изменением размера структуры данных.

---

## ✅ Выполненные задания

| № | Задание | Реализация | Статус |
|---|---------|------------|--------|
| 01 | Класс `HashTable` с методами `insert`, `search`, `delete` | Коллизии обрабатываются методом цепочек (списки в ячейках) | ✔️ Готово |
| 02 | Метод `resize` | Размер таблицы увеличивается вдвое, все элементы перераспределяются с пересчётом индексов | ✔️ Готово |
| 03 | Функция `simple_hash(s)` | Возвращает сумму ASCII-кодов символов строки (`sum(ord(c) for c in s)`) | ✔️ Готово |
| 04 | Класс `StringHashDict` | Использует `simple_hash` для хранения пар «строка — хеш», реализованы методы `add` и `get_hash` | ✔️ Готово |
